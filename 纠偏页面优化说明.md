# 纠偏页面优化说明

## 📋 优化内容

### 1. 整合搜索/筛选区域
- **位置**：模态框顶部的淡蓝色渐变区域
- **组件**：
  - 🔍 **搜索关键字**：输入文本过滤表格内容
  - 📌 **排除原样式**：勾选不想显示的 Word 原始样式（如 Normal、Heading 1）
  - 🎯 **排除目标规范**：勾选不想显示的目标规范（如文档标题、正文）
  - ⚡ **批量纠偏**：6个按钮快速将当前筛选结果批量设置为指定规范

### 2. 联动逻辑
- **搜索 + 排除样式 + 排除目标** 三者同时生效
- 表格实时显示过滤后的结果数量
- 批量纠偏按钮仅对**当前筛选结果**生效，不影响被过滤掉的段落

### 3. 布局优化
- **横向排列 + 自动换行**：所有复选框和按钮横向铺满后自动换行
- **统一间距**：使用 `gap-2` 和 `mb-4` 保持视觉一致性
- **卡片化设计**：整个筛选区域采用圆角卡片 + 渐变背景，消除突兀感
- **紧凑高度**：单个区域垂直紧凑，不占用过多空间

### 4. 移除的元素
- ❌ 删除了重复的"批量纠偏"按钮（原本有两处）
- ✅ 保留一个统一的批量纠偏区域，位于筛选区域底部

## 🎨 视觉改进
- 使用 `from-blue-50/50 to-indigo-50/50` 渐变背景
- 边框颜色统一为 `border-blue-100`
- 图标前缀：🔍 🎯 📌 ⚡ 增强识别度
- 清除按钮：每个筛选项右侧提供快速清空功能

## 🔧 技术实现
- 新增状态：`searchKeyword`、`excludedTargets`
- 新增函数：`handleBatchCorrect(targetKey)`
- 过滤逻辑：三重过滤（原样式 + 目标规范 + 关键字）
- 模态框宽度：900px → 1000px（容纳更多内容）
- 表格高度：400px → 350px（为筛选区域腾出空间）

## 📝 使用流程
1. 上传 Word 文档后，系统自动打开纠偏模态框
2. 在顶部筛选区域：
   - 输入关键字缩小范围
   - 勾选"排除原样式"隐藏不需要的原始样式
   - 勾选"排除目标规范"隐藏不需要调整的目标类型
3. 查看表格，确认当前显示的段落数量
4. 点击"批量纠偏"按钮，快速将筛选结果设为指定规范
5. 或在表格中逐行手动调整
6. 点击"确认并开始格式化"应用修改

## 🎯 典型场景
**场景 1**：只想调整正文段落
- 勾选"排除目标规范" → 除"正文"外全选
- 点击批量纠偏 → 选择"正文"

**场景 2**：标题样式识别错误，批量修正
- 输入关键字"第一章"
- 点击批量纠偏 → "一级标题"

**场景 3**：排除表格/引用等特殊样式
- 勾选"排除原样式" → 选中 "Table"、"Quote"
- 只调整文档主体内容
